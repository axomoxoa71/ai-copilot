# src Instructions

This folder contains the React + TypeScript frontend source.

## Key Files

- `main.tsx` — Entry point, mounts the React app in `#root`
- `App.tsx` — Root component with routing and navigation shell
- `App.css` — Application-level CSS (layout, components)
- `index.css` — Global CSS reset and CSS custom properties
- `vite-env.d.ts` — Vite type declarations

## Conventions

- All source files must use TypeScript (`.ts` or `.tsx`).
- Shared, reusable components go in `src/components/`.
- All UI text must be in English.
- Follow requirements documents in `requirements/` when implementing screens.
